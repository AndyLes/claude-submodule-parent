# E3 хрестокрилий ракетний інтерсептор — друк, збірка, пуск

> Числа: [`e3_config.py`](e3_config.py) · ТТХ: `python -m cuas.design.report_e3` · модель: [`cad/e3_missile.scad`](../../cad/e3_missile.scad).
> **Безпека:** ніс — інертна маса, **без БЧ**. Пуск/політ — на інертних мішенях, у legally-authorized умовах.

## ТТХ (рекомендована зв'язка)
AUW ~1249 г · навантаження на крило ~89 г/дм² · зрив ~45 км/год · **схід з труби ~57 км/год** · **макс. ~177 км/год** · енергія тарана **~1509 Дж** · планування ~4 км з 500 м · хвостовий обсяг 0.64 (стійкий) · CoG +38 мм (центрується акумом).

⚠️ **Проп критичний:** ваш **9×2 (крок 2″) НЕ ЛІТАЄ** — стеля 46 км/год ≤ зрив 45 → падає з труби. Рекомендовано **7×6** (крок 6″).

## Друк (PETG)
```
mkdir -p cad/out
openscad -D 'part="nose"' -o cad/out/e3_nose.stl cad/e3_missile.scad
openscad -D 'part="body"' -o cad/out/e3_body.stl cad/e3_missile.scad
openscad -D 'part="tail"' -o cad/out/e3_tail.stl cad/e3_missile.scad
```
Слайсер: PETG, 3–4 периметри, заповнення 20–30% (gyroid), шар 0.2. Секції друкувати вертикально по осі (міцність), стик — клей/муфта. Хрест-крила — інтегровані в секції body/tail (виходять зі стіни фюзеляжу).

## Порядок збірки (максимально спрощено)
1. **Стек:** F405 Wing + 50A ESC + мотор (пушер) на хвостову моторам́у (M3×16). Серва (4× MG90S) на хвостові керма.
2. **Секції:** склеїти ніс+body+tail (муфти), протягнути кабелі. Ніс — інертна маса + Foxeer у носовий отвір (об'єктив вперед).
3. **Акумулятор:** 4S 6500 у відсік через бічний люк; **зсувати для CoG** (~+30…+40 мм; ніс злегка важчий — норм для ракети).
4. **Баланс:** підвісити за точку CoG; має бути стабільно, ледь носом униз.

## ArduPlane (F405 Wing)
- Frame/тип: **звичайний літак**; кермо-мікс під **хрест** (4 виходи: комбінація elevon/rudder на 4 площини).
- **Failsafe = RTL/Circle** при втраті лінку.
- Політ: ручний FPV (Foxeer+VTX+окуляри); автономний термінал — окремо, коли додасте бортовий комп (Pi/Jetson) + оцифровку.

## Пуск із труби / катапульти
- Схід ≥ **~57 км/год** (над зривом ×1.25) — бунджі/пневмо/резина у контейнері.
- Труба ID ≥ розмах хрест-крил; напрямні — по кінцях крил/фінів.
- Після сходу: газ на max, набір кута, перехід у горизонт → наведення на ціль.

## Чек перед пуском
- [ ] Проп **високого кроку** (5.5–6″), НЕ 9×2.
- [ ] CoG у точці; керма рухаються у правильний бік (хрест-мікс перевірено).
- [ ] Failsafe спрацьовує; мотор реверс-тяги немає.
- [ ] Ніс інертний; жодних БЧ.
