# E3 інтерсептор — складане монопланне крило, тубусний старт, АВТОНОМНИЙ

> Числа: [`e3_config.py`](e3_config.py) · ТТХ: `python -m cuas.design.report_e3` · модель: [`cad/e3_missile.scad`](../../cad/e3_missile.scad).
> **Безпека:** ніс — інертна маса, **без БЧ**. Пуск/політ — на інертних мішенях, у legally-authorized умовах.

## ТТХ (компактний тубусний; проєкт 6S / тест 4S) — площа крила з ГЕОМЕТРІЇ
AUW ~1103 г · **крило 4.48 дм²** · навантаження **246 г/дм²** · зрив ~71 км/год · схід з труби ~89 км/год · **радіус розвороту ~40 м** (широкий — ставка на pixel-lock, не «крутіж») · хвіст 0.54 (ок) · CoG +42 мм · труба **ID ≥116 мм**.

| Режим | Макс. швидкість | Летить | Таран |
|---|---|---|---|
| **6S (проєкт)** | **~199 км/год** | ✅ | **~1687 Дж** |
| **4S (маіден)** | ~133 км/год | ✅ | — |

> **Чесно:** це Switchblade-клас — швидкий, високе навантаження, широкий розворот; влітає в ціль по бортовому наведенню. Площа крила тепер **обчислюється з геометрії CAD** (тест-запобіжник проти минулої розбіжності 22↔4.8 дм²).

**Чому моноплан:** більше крило → тугіший розворот → **ефективніше перехоплення**; нижчий зрив; стандартний мікс (простіше за хрест). Складається для труби.

## АВТОНОМІЯ (обов'язково)
- **Pi Zero 2W + цифрова камера** у носі → **pixel-lock** веде на ціль автономно (стійко до РЕБ — команда дана поза зоною глушіння).
- Оператор: лише **OK на пуск / ABORT**. Foxeer-аналог — резервний ручний FPV.
- Софт: [`cuas/effectors/e2_ram/companion.py`](../effectors/e2_ram/companion.py) — vision lead-pursuit → MAVLink offboard (готовий).
- Звʼязок: Pi ↔ F405 Wing по UART (MAVLink); живлення Pi від BEC 5В.

## BOM (проєкт 6S / тест 4S; ✅ = ваше)
| Вузол | Специфікація |
|---|---|
| Мотор (пушер) | **~1200KV 6S-сумісний** — 4S ~133, 6S ~199 км/год |
| Проп | **7×6** (не 9×2!) |
| ESC | **50A, 6S-сумісний** ⚠️ (для 4S-тесту ваш ОК) |
| **Автономія** | **Pi Zero 2W + камера** (обов'язково) + BEC 5В |
| Керма | **3× MG90S** (2 елерони + елеватор) |
| FC | ✅ **F405 Wing** (ArduPlane) |
| FPV-резерв | ✅ **Foxeer аналог** |
| Акумулятор | **6S 3500** (повна, ~500 г — слимкіший фюзеляж 50 мм) + ✅ **4S** (тест) |
| Крило | друк PETG, **складане** (2 половини, пін-шарнір + торсійна пружина + стоп) |
| Хвіст | малі фіксовані фіни (в трубу) |

## Реальна геометрія (перевірено числом)
- **Корпус 52×46 мм** — розмір задає **батарея 6S 3500 (43×38)**; внутр. 47×41 → всі компоненти влазять (перевірка: `component_fits_body`).
- **Крило — справжній NACA ~10% профіль** (не пласка пластина), несуча — **CF-пруток Ø8** у каналі на 30% хорди; хорда 72→48, піврозмах 373 → площа **4.48 дм²** (= геометрія).
- **Складання: корені L/R РОЗНЕСЕНІ по довжині** (stagger 42 мм) → половини не зіштовхуються; кожне — пін-шарнір + защіпка-детент + стоп 90°.
- **Складена обвідна ≤ 112 мм → труба 115 мм** (перевірка: `folded_tube_id_required_mm`). Хорда 72 лягає впоперек.
- **Розкладка (ніс→хвіст):** камера+Foxeer+баласт (ніс) · Pi Zero 2W · FC+RX · **батарея (мід)** · ESC · серва (елеватор+руль) · мотор (пушер). CoG **+20 мм** (на 10 мм перед крилом → стійко). `part="layout"` показує все на місцях.
- **Защіпка розкриття (`part="latch"`)** — пружинний детент-плунжер: крило при розкритті само відводить скошений ніс, на 90° отвір крила співпадає → пружина заганяє плунжер у замок (**авто-фіксація**). Натиснути хвостик → відвести → скласти. Куповане: пружина стиску ~Ø3×10 мм (~1 Н). *(Друкований плунжер може підклинювати — надійніше сталевий штифт Ø3 як детент.)*
- ⚠️ **Тонкий фіт (зазори шарніра/защіпки, посадка секцій) підтвердити ТЕСТ-ДРУКОМ** — у CAD не перевіряється абсолютно.

## Корпус: rounded-rect + висувний лоток (простіша збірка/сервіс)
Плоскі стінки → **усе обладнання на одному лотку** (FC+ESC+Pi+RX+ремінь акума), що **висувається через верхній люк** — замість вигнутих кріплень. Друк без склепінь/підпор. Швидкість prop-limited → опір «коробки» коштує лише ~кілька % дальності. Пуск — **квадратна труба/рейка** (простіша за точну круглу).

+ куповані: **CF-пруток Ø8 ×2** (лонжерони), пін-стрижень Ø2.5 ×2 (шарнір). PETG, 3–4 периметри, 20–30% gyroid, 0.2 мм.

## Складане крило + пуск із труби (шарнір змодельовано)
- У корені кожної половини — **пін-шарнір** (барабан + пін Ø2.5, вісь складання = товщина крила); крило **складається назад** уздовж фюзеляжу.
- **Торсійна пружина** у гнізді барабана **розкриває** крило на виході; **стоп** фіксує польотне положення (90°); защіпка-фіксатор утримує.
- **Труба ID ≥ 120 мм** (визначено фіксованими фінами; складене крило вужче).
- Схід ≥ ~41 км/год (над зривом ×1.25) — бунджі/пневмо.

## Порядок збірки
1. **Стек:** F405 Wing + 50A ESC + мотор (пушер, M3×16). 3× MG90S: 2 елерони (у крилі) + елеватор (хвіст).
2. **Автономія:** Pi Zero 2W + камера в носовий відсік (об'єктив вперед через отвір), UART до FC, живлення BEC. Foxeer поруч (резерв).
3. **Секції:** ніс+body+tail склеїти; протягнути кабелі. Ніс: інертна маса + комп + камери.
4. **Крило:** шарнір у корені + пружина розкриття + фіксатор; елерони на шарнірах, горни, тяги MG90S.
5. **Акумулятор:** у відсік через люк; **зсувати для CoG** (~+40 мм, ледь носом униз).
6. **Баланс + напрям керм** перевірити до пуску.

## ArduPlane (F405 Wing)
- Тип: **звичайний літак** (елерон+елеватор — стандартний мікс, не хрест).
- **Failsafe = RTL/Circle**.
- Автономний термінал через companion (Pi → MAVLink GUIDED); оператор OK/ABORT.

## Міцність (перевірено — `python -m pytest cuas/tests/test_structures.py`)
- **Несуча крила = вуглепластиковий пруток Ø8 мм** (2 шт, по консолі); друкована оболонка лише формоутворює (суцільне крило важило б ~613 г).
- Лонжерон 8 мм CF тримає **~14 g**; проєктний ліміт **10 g** (запас 1.41).
- ⚠️ **Обмежити перевантаження ≤10 g:** у ArduPlane виставити ліміт G / зменшити хід елеватора — на 199 км/год повне кермо дало б ~37 g і зламало крило.
- Старт із труби ~5.5 g / 64 Н — фюзеляж тримає легко.
