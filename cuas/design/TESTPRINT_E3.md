# E3 — чек-лист першого тест-друку (допуски)

> CAD перевіряє геометрію, але НЕ абсолютні допуски. Це список того, що виміряти на
> першому фізичному друці, і який параметр у `cad/e3_missile.scad` правити, якщо не сходиться.
> Друкувати спершу **лише вузли** (шарнір+защіпка, стик, бей), не весь планер.

## Пріоритет 1 — механізм крила (найризикованіше)
| Перевірка | Норма | Якщо не так → правити |
|---|---|---|
| Пін Ø4 у вушку/вилці | ковзає без люфту | `pin_d` (бор 4.2) — калібрувати ±0.1 під принтер |
| Вилка крила обіймає вушко фюзеляжу | зазор ~0.3 мм з боків | `kn_h` / рознос вилки (`sy*8`) |
| Складання → розкриття | вільно 0…90°, без тертя | зазори вушка; орієнтація друку |
| **Стоп на 90°** | крило стає рівно поперек (90.0°) | позиція «deploy stop» у `wing_knuckle` |
| **Защіпка авто-клацає** | на 90° плунжер сам заходить у замок | співвісність lock-hole (`+5` offset) ↔ детент-seat |
| Защіпка тримає складання | не відкладається під рукою | глибина замка / жорсткість пружини |
| Рознесені крила НЕ б'ються | складені half повз half | `stagger` (зараз 42) |
| Складена обвідна | ≤ 112 мм (труба 115) | `wing_rc` (хорда) |

## Пріоритет 2 — несуча й посадки
| Перевірка | Норма | Правити |
|---|---|---|
| CF-пруток Ø8 у канал крила | входить щільно, вклеюється | `spar_d` (8.2) |
| Шпиг стику ніс/корпус/хвіст | плуг у гніздо, співвісно | `spig`; зазор гнізда (+1.2) |
| Лоток крізь верхній люк | висувається вільно | люк `BW-8` ↔ лоток `tw` |
| Батарея 6S3500 (43×38) у бей | ~2 мм зазор з боків | `BW`/`BH` (52/46) |
| Бей FC/ESC/Pi | компоненти сідають | розміри в `COMPS`/`tray` |
| Мотор M3 16 мм | збігається болт-патерн | `motor_mount` отвори |
| Обʼєктив у ніс | пролазить | `cam_d` (10) |

## Пріоритет 3 — маса й міцність
| Перевірка | Норма | Дія |
|---|---|---|
| Маса корпусу+крил | крило ~100 г, фюзеляж ~185 г (бюджет) | якщо важче → менше периметрів/інфіл; перерахувати AUW |
| Промінь/стінка каналу лонжерона | без розшарування | більше периметрів; сушити PETG |
| Тест на згин крила (з CF) | тримає ~10 g еквівалент (навантажити ~11 кг на розмах статично з запасом) | більший CF-пруток якщо гнеться |

## Порядок
1. Друк **вузол шарніра+защіпка** (ніс частини вушка + корінь крила) → відпрацювати fold/deploy/lock, калібрувати `pin_d`/зазори.
2. Друк **секція-стик** (два коротких відрізки) → перевірити шпиг.
3. Друк **бей+лоток** → батарея + лоток.
4. Аж тоді — повний планер.
5. Записати фактичні маси → оновити `COMPONENTS` у `e3_config.py` → перерахунок ТТХ.
