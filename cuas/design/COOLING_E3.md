# E3 — охолодження (електроніка / мотор / батарея)

> Числа: `python -m cuas.design.report_e3` (блок THERMAL) · модель тепла: [`thermal.py`](thermal.py) · тести: `pytest cuas/tests/test_thermal.py`.
> **Безпека:** ніс — інертна маса, **без БЧ**. Тест — на інертних мішенях, у legally-authorized умовах.

## Головне — де насправді ризик
Для тубусного пушер-інтерсептора **політ теплово легкий**, а **пре-старт — критичний**:

- **У польоті** пушер-проп працює **витяжкою** — тягне повітря крізь фюзеляж. Тепла мало виносити (мотор обдувається зовні), швидкості вистачає навіть на сході. Легко.
- **Пере-стартом** планер сидить у **ЗАКРИТІЙ трубі**: мотор/ESC вимкнені (нуль обдуву), а авіоніка гріє глухий об'єм. **Аналоговий VTX** — найгарячіший дрібний вузол; за хвилини він перегрівається. **Це і є справжній ліміт.**

## Рішення — 3 контури

### 1. Авіоніка в польоті (пасивний наскрізний обдув)
- **2 втоплені NACA-входи** по бортах (z≈135, перед крилом). Втоплені (submerged) — **не виступають → переживають трубу**, ловлять набігний потік одразу після сходу.
- Потік іде **ніс→корму**: омиває FC (z150) → батарею (z200–330) → **ESC (z380)**.
- **Аутлет-луври** (3 наскрізні Ø5 з боків, z≈392–414, прямо біля ESC) → викид у зону розрідження за корпусом; пушер-проп підсмоктує.
- CAD: `naca_inlet()` ×2, `exhaust_louvers()`. Площа входів ≈0.97 см² — з запасом (на сході ΔT повітря 8.5 °C).

### 2. Мотор (пушер — обдув зовні + брізер)
- Пушер-проп тягне **набігний потік по хвостовому конусу через мотор** — 2807-клас на 40+ м/с обдуву охолоджується легко (на мультикоптері ті самі мотори тримають 300 Вт із меншим обдувом).
- **Брізер-слоти** у бобишці мотора (`motor_cool_slots()`, 6×Ø3.5) — проп прокачує повітря крізь дзвін/статор.
- ESC — **6S-сумісний, у потоці бею** (не замуровувати в піну).

### 3. Пре-старт соук — ГОЛОВНЕ (у трубі обдуву нема)
Без заходів VTX перегріє VTX за **~3 хв**. Тому:
1. **Гейтити VTX** — тримати аналоговий передавач **вимкненим/на min потужності** в трубі; вмикати на команду пуску (він резервний FPV, у трубі не потрібен).
2. **Pi в ідлі** до команди — vision-навантаження вмикати вже в повітрі (ціль однаково лочиться після сходу).
3. **Алюмінієвий тепло-розподільник (~60 г)** на термопасту під VTX+Pi — додає теплову масу.
   Разом → **соук ~23 хв** (замість 3). Достатньо для чергування «армованим».
4. **Операційний ліміт дуель-часу** в трубі — тримати в звіті; якщо все ON без розподільника — **не довше ~3 хв**.

> **Чому не активний кулер:** місія коротка (спринт ~40–60 с), маса критична, а пушер уже дає наскрізний потік. Вентилятор/радіатор — зайва маса й точка відмови. Пасив + гейтинг VTX закривають задачу.

## Якщо перепрофілювати в **loiter** (довгий політ)
Тоді батарея й ESC гріються тривало → знадобиться: більші входи, обдув батареї, можливо термопрокладка ESC на скін. Для **інтерсептора** (короткий спринт) — не треба (перевірено: батарея 35 °C).
