# E3 пусковий launcher — ПНЕВМАТИЧНИЙ (рішення обґрунтоване)

> Розрахунок: `python -m cuas.design.launch` · тести: `pytest cuas/tests/test_launch.py` · друк: [`cad/e3_launcher.scad`](../../cad/e3_launcher.scad).

## Чому пневматика (а не бунджі)
Компактний планер має високе навантаження → схід **~89 км/год (24.8 м/с)**, енергія **~340 Дж**. Пускове g **ОСЬОВЕ** (крила складені, фюзеляж у стиску) — окреме від польотного ліміту 10 g (той для аеро-згину крила в маневрі). Осьове ~20 g планер тримає легко (як Switchblade).

| Труба | Осьове g | Бунджі пік | Пневмо тиск (130мм бор) |
|---|---|---|---|
| 1.3 м | 24 g | ~43 g | 0.10 бар |
| **1.6 м** | **~20 g** | ~36 g | **0.19 бар** |
| 2.0 м | 15.7 g | ~28 g | 0.15 бар |

- **Пневматика = постійна сила** → пік = середнє (кероване). Бунджі спадає → пік ~1.8× → зайве навантаження й гірша повторюваність.
- **Повторюваний схід** (тиск = швидкість) — критично для автономного наведення.
- **Труба ~1.6 м**, тиск **~0.2 бар** (низький; 130мм повнопрохідний поршень), швидкий перезаряд.

## Система
- **Труба:** квадратна off-the-shelf (алюм. профіль/ПВХ) **ID ~72 мм**, довжина **~1.3 м**, під кутом ~30–45°.
- **Казенник** (друк): герметизує зад, впуск ¼″, кріпиться болтами; за ним — **бак** ~1–2 л.
- **Клапан:** QEV (quick-exhaust) або великий кульовий / спринклерний соленоїд (для автоматизації по команді C2).
- **Поршень-sabot** (друк): ущільнення (O-ring/пінолента), штовхає хвіст планера, зупиняється на дульній напрямній (планер вилітає вільно).
- **Дульна напрямна** (друк): центрує планер, ловить поршень.
- **Живлення:** ручний/ножний насос або компресор + редуктор (~1 бар); бак тримає кілька пусків.

```

```
+ куповані: квадратна труба ~75×75, бак, клапан, редуктор, насос, O-ring.

## Цикл пуску
1. Завантажити планер (**крила/хвіст складені**) + поршень за ним.
2. Накачати бак до заданого тиску (= задана швидкість сходу ~13 м/с).
3. Оператор **OK** → відкрити клапан → постійні ~6.6 g → схід 47 км/год.
4. У повітрі: крила/хвіст **розкриваються пружиною**, пушер розкручується → автономний термінал (pixel-lock).
5. Перезаряд: підкачати бак, зарядити наступний.

## ⚠️ Зчеплення (треба вирішити в планері)
Компактна труба ID 72 мм вимагає, щоб **хвіст теж складався** (як крила) — інакше велике оперення не влізе. Варіанти:
- **A.** Складаний хвіст (як Switchblade) — компактна труба, але +механізм.
- **B.** Більша труба (~130 мм) з малим фіксованим хвостом — простіше, але гірша стійкість при великому крилі.
- **C.** Рейка-катапульта замість труби — планер не складає хвіст узагалі, але launcher громіздкіший/невсепогодний.
Рекомендація: **A** (складаний хвіст) для справжнього тубуса; наступна правка планера.
