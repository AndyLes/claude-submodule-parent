# E1 — багаторазовий сітка-дрон (BUILD)

Проти розвід-квадрокоптерів. Ловить сіткою, **повертається** (витрата/ураження ≈ картридж $20–50 + знос).

## BOM (~$1,100 планер + сітколет + компут)
- несучий 7–10″ квадро
- гімбал-камера + **Pi 5 / Coral** (авто-наведення)
- **сітколет** (пружина/CO₂), картридж-сітка $20–50 (ремонтопридатний)
- парашут або м'яка посадка для повернення неушкодженим
- FC з ArduPilot (offboard/GUIDED), ELRS RX

## Приймання (інертно, полігон)
1. **Захоплення+повернення:** інертний квадро (зависання/повільний рух); C2→OK; E1 зближується, скидає сітку, ціль заплутана й опускається, E1 **RTL неушкодженим**. Порахувати вартість/ураження.
2. **Повторюваність:** ≥5 циклів «виліт → захоплення → повернення → перезарядка сітки» одним апаратом; зафіксувати середній час перезарядки та відмови.

> Софт: `cuas/effectors/e1_net/mission.py` (умова скиду сітки) + `companion.py` (approach + trigger + RTL, перевикористовує guidance від E2).
