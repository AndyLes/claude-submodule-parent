# E2 — наддешевий авто-таран (BUILD)

Проти швидкого FPV на ближньому рубежі. Одноразовий, але дешевий (обмін 1,5–7:1).

## BOM (~$150–200/шт)
- 5″ рама (карбон)
- 4× мотори ~2306/1900KV + 4-в-1 ESC
- FC з **ArduPilot** (offboard/GUIDED), ELRS RX
- 6S 1300 mAh LiPo (висока C)
- дешева камера (глобальний затвор бажано) + **Pi Zero 2W / Coral** (бортове бачення)
- місце під інертний імітатор БЧ
- **без VTX / без пілота** (наведення автономне)

## Приймання (інертно, полігон)
1. **Швидкість/набір:** макс. горизонт. швидкість ≥ 70 м/с (252 км/год) або зафіксувати факт для моделі M0; набір ≥ 25 м/с.
2. **Авто-перехоплення:** інертна FPV-мішень; C2→OK; E2 автономно (бортове бачення) уражає без пілота; ABORT з консолі відводить апарат. Записати % влучань за N спроб.

> Софт наведення: `cuas/effectors/e2_ram/guidance.py` (+ `companion.py` для MAVLink). БЧ/заряд — поза цим репо.
